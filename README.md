# ranjithvelpur.github.io

Personal profile website for Ranjith Velpur — Senior Java Backend / Cloud Engineer.

## Deploy to GitHub Pages

```bash
# 1. Create a repo named exactly: ranjithvelpur.github.io
#    (Settings → Pages → Source: Deploy from branch → main / root)

# 2. Initialize and push
cd ranjithvelpur.github.io
git init
git add .
git commit -m "Initial profile site"
git remote add origin https://github.com/ranjithvelpur/ranjithvelpur.github.io.git
git push -u origin main
```

Your site will be live at **https://ranjithvelpur.github.io** within ~1 minute.

## Customize

- Update email/LinkedIn/GitHub links in `index.html` (search for `example.com` and `ranjithvelpur`)
- Add your actual employment dates (search for `YYYY`)
- Add a profile photo: place `photo.jpg` in the folder and add `<img src="photo.jpg">` in the hero section
